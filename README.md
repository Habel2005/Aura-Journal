# Awwwards Catalyst

A Next.js & Genkit starter template designed to build visually stunning, Awwwards-worthy websites. This project provides a solid foundation for creating immersive digital experiences with a focus on sophisticated design, fluid animations, and interactive elements.

## ✨ Core Features

-   **Dynamic Visual Showcase**: Present creative projects with bespoke, interactive visual effects, potentially powered by WebGL and advanced graphics.
-   **Seamless Navigational Flow**: Implements sophisticated scrolling and page transitions using libraries like Lenis for an exceptionally fluid and engaging user journey.
-   **Creative Project Deep Dive**: Allows users to explore individual projects through rich media, detailed descriptions, and interactive elements.
-   **AI-Powered Features**: Integrated with Google's Genkit to enable generative AI capabilities within your application.
-   **Component-Based Architecture**: Built with a rich set of UI components from `shadcn/ui` for rapid and consistent development.

## 🎨 Design Philosophy

The design is guided by a set of modern, sophisticated principles:

-   **Color Palette**: Sophisticated, often monochromatic palettes with strategic use of vibrant accent colors to emphasize interactive elements.
-   **Typography**: Contemporary, bold display typefaces for headlines and clear, legible sans-serifs for body content.
-   **Layout**: Dynamic, grid-breaking layouts with ample whitespace to frame and enhance visually rich content.
-   **Animations**: Cutting-edge, performant animations and transitions, including smooth scrolling and micro-interactions for an immersive experience.

## 🛠️ Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
-   **AI Integration**: [Genkit (Google)](https://firebase.google.com/docs/genkit)
-   **Deployment**: Pre-configured for [Google App Hosting](https://cloud.google.com/app-hosting) (`apphosting.yaml`)

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine.

### Prerequisites

-   Node.js (v18 or later)
-   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/awwwards-catalyst.git
    cd awwwards-catalyst
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## ☁️ Deployment

This project includes an `apphosting.yaml` file, making it ready to be deployed on Google Cloud's App Hosting service.

For more details on deployment, refer to the [Google App Hosting documentation](https://cloud.google.com/app-hosting/docs/deployment).
